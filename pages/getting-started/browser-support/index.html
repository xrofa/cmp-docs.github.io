    Browser Support

    Loader.js is our standard script tag. It supports browsers starting from these versions:

    Browser Version
    Chrome 37
    Edge 15
    Firefox 41
    Internet Explorer 11
    Opera 31
    Safari 11
    Yandex 17.6
